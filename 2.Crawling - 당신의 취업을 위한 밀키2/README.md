# 취업 분석 프로젝트 (Job Recruitment Analysis)

이 프로젝트는 채용 공고 데이터를 수집, 분석하여 취업 전략을 개발하기 위한 도구를 제공합니다.

## 📋 프로젝트 개요

사람인 웹사이트에서 채용 정보를 크롤링하여 기업 정보, 우대사항, 지역 분포, 급여 정보 등을 분석하는 프로젝트입니다. 데이터 시각화를 통해 취업 시장의 트렌드를 파악하고 개인 맞춤형 취업 전략을 수립할 수 있습니다.

## 🛠️ 주요 기능

1. **채용 정보 크롤링**
   - 사람인에서 키워드 기반 채용 정보 수집
   - 기업 정보(연봉, 매출액, 영업이익 등) 크롤링

2. **우대사항 키워드 추출**
   - 채용 공고에서 우대사항 텍스트 추출
   - 키워드 시각화 (워드 클라우드)

3. **데이터 시각화**
   - 지역별, 학력별 분포 파이 차트
   - 기업별 연봉, 매출액, 영업이익 비교 그래프
   - 평균 연봉 기준 기업 순위 시각화

4. **GUI 인터페이스**
   - 사용자 친화적인 GUI로 손쉬운 데이터 분석
   - 파일 업로드 및 결과 확인 기능

## 🗂️ 프로젝트 구조

```
├── job_info_GUI_ver1.0.5.py         # 메인 GUI 프로그램
├── Make_word_Text_.py               # 우대사항 키워드 추출 모듈
├── crawl_company.py                 # 기업 정보 크롤링 모듈
├── crawl_recruitment.py             # 채용 공고 크롤링 모듈
├── vizual_function.py               # 데이터 시각화 함수 모듈
└── extracted_preference_keywords.txt # 추출된 우대사항 키워드 파일
```

## 📊 데이터 분석 흐름

1. **데이터 수집**
   - 검색어 입력 → 채용공고.csv 생성
   - 채용공고.csv를 기반으로 기업정보.csv 생성
   - 우대사항 키워드 추출 → extracted_preference_keywords.txt 생성

2. **데이터 병합 및 전처리**
   - 채용공고.csv와 기업정보.csv 병합
   - 데이터 필터링 및 정제
   - 타입 변환 (문자열 → 숫자)

3. **데이터 시각화**
   - 막대 그래프: 연봉, 매출액, 영업이익 기준 기업 순위
   - 파이 차트: 지역별, 학력별 분포
   - 워드 클라우드: 우대사항 키워드 분석

## 📈 취업 전략

1. **지역 고려**
   - 서울 지역 채용이 압도적으로 많음 (80% 이상)
   - 지방 취업 시 경쟁률이 낮을 수 있음

2. **학력 요건**
   - 학력무관과 대졸 이상이 비슷한 비율
   - 초대졸 이상도 상당수 존재

3. **우대사항 공략**
   - 프로젝트 경험이 가장 중요
   - 관련 기술 능숙도 및 자격증 준비
   - 커뮤니케이션 능력 강조

4. **기업 선정 전략**
   - 연봉: 4500만원 이하 현실적 고려
   - 사원수: 100명 이상 안정성 고려
   - 매출액: 동종업계 상위 10개 기업 타겟팅
